---
title: Run a monthly client report
description: A repeatable monthly routine to refresh data and deliver a branded performance report.
---

A tight monthly cadence keeps clients informed and surfaces the next month's work.

<Steps>
<Step>
### Refresh the data
Make sure GSC/GA4 have synced and run a fresh [crawl](/docs/getting-started/first-crawl-and-audit)
so audits and on-page data reflect the current site.
</Step>
<Step>
### Re-run AI visibility (optional)
For clients on an AI-visibility plan, re-run the [audit](/docs/guides-track/ai-visibility)
so the report shows month-over-month movement.
</Step>
<Step>
### Generate the report
From [Reports](/docs/guides-report/reports), generate the monthly performance
report (and the AI-visibility report if relevant).
</Step>
<Step>
### Review & annotate
Skim the rankings movements and published content; make sure the numbers tell the
story you want.
</Step>
<Step>
### Share it
Send the [branded share link](/docs/guides-report/sharing-and-branding) (password
it if needed).
</Step>
</Steps>

<Callout type="info">
  Scheduled monthly reports can generate automatically — use this playbook to
  review and send, rather than building from scratch each month.
</Callout>
