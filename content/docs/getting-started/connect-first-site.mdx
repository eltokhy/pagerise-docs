Getting a site into Pagerise takes a few minutes. The onboarding wizard walks you
through it, or you can do the steps manually.

## Use the onboarding wizard

New workspaces open on a guided **Onboarding** flow that covers the essentials:

<Steps>
<Step>
### Add the domain
Enter the site's URL. Pagerise normalizes it and creates the site record.
</Step>
<Step>
### Assign a client
Attach the site to a new or existing client, and add a few target keywords.
</Step>
<Step>
### Seed a crawl
Kick off the first crawl so the app has pages to analyze.
</Step>
<Step>
### Run an initial audit
Pagerise runs the on-page audit and surfaces your first Tasks.
</Step>
</Steps>

## Connect your data sources

The richest results come from connecting the site's accounts. In
**Settings → Google accounts** and **Settings → Integrations**:

- **Google Search Console (GSC)** — powers Search Insights (clicks, impressions,
  positions, queries) and URL inspection.
- **Google Analytics 4 (GA4)** — traffic, conversions, and AI-assistant referrers.
- **Google Business Profile (GBP)** — post to the listing and manage locations.
- **WordPress** — publish content and apply fixes directly to the live site. Use
  the one-click plugin install or connect manually with an application password.

<Callout type="info">
  You can use Pagerise without WordPress — you'll still get crawling, audits,
  Search Insights, AI visibility, and reports. WordPress just enables one-click
  publishing and fixes.
</Callout>
