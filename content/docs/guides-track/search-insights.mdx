**Track → Search Insights** is the Google Search Console (GSC) view of a site —
how it performs in organic search and which keywords drive it. Connect GSC in
[Settings → Google accounts](/docs/settings/integrations) to populate it.

## Search Visibility

- **Traffic** — clicks, impressions, CTR, and average position over time, by query
  and landing page.
- **CTR** — click-through-rate analysis to find page-one results that under-earn
  clicks (a snippet-rewrite opportunity).
- **Google Updates** — overlays known algorithm updates so you can see their
  impact on the site.
- **Acquisition** — traffic-source breakdown.
- **Cannibalization** — queries where multiple pages compete, splitting rankings.

## Strategic Keywords

- **Topics** *(Pro)* — semantic topic clusters for research and planning.
- **Focus** — keywords you're actively tracking.

## All Keywords

Filtered keyword lists: **All**, **New** (newly ranking), **Low-hanging** (quick
wins near page one), **Lost** (recently dropped), and **Branded**.

<Callout type="info">
  Search Insights reflects GSC data, which lags by a couple of days. Pair it with
  [AI Visibility](/docs/guides-track/ai-visibility) for the AI-search side of the
  picture.
</Callout>
