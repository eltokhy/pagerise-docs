**Run → Skills** are one-click AI tasks you run on a site. Each Skill reads the
latest crawl, performs its analysis, and returns **scored findings** (issues and
opportunities) you can act on.

## Available Skills

- **AI Visibility audit (`ai_seo`)** — scores citability, brand mentions across
  platforms (YouTube, Reddit, Wikipedia, LinkedIn), AI-crawler access
  (robots.txt + llms.txt), and structured-data signals. The deep companion to the
  [AI Visibility](/docs/guides-track/ai-visibility) tracker.
- **Content optimization** — readability, keyword, and structure recommendations
  for existing pages.
- **Competitor gap analysis** — topics and keywords competitors cover that the
  site doesn't.
- Additional Skills (on-page audit, CRO, programmatic-SEO discovery) appear as
  they ship.

## Running a Skill

<Steps>
<Step>
### Pick a Skill
Open **Skills** and choose the task to run.
</Step>
<Step>
### Let it analyze
It runs as a background job against the latest crawl snapshot.
</Step>
<Step>
### Review findings
Results come back as scored dimensions plus specific issues and opportunities.
</Step>
</Steps>

<Callout type="info">
  Skills read the latest crawl. For current results, crawl the site first.
  Skill runs draw from your workspace AI quota — see
  [Plans & limits](/docs/reference/plans-and-limits).
</Callout>
